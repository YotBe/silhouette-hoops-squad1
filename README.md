🎮 Silhouette Hoops Squad
NBA Player Guessing Game - Identify basketball legends from disguised silhouettes and video clips across different eras. Fast, addictive mobile-first gameplay with multiplayer party mode.

https://silhouette-hoops-squad1.vercel.app/

✨ Features
Core Gameplay: Guess NBA players from silhouettes, blurred clips, or altered videos

Multiple Modes: Casual, Daily Challenge, Endless, MVP/Legend

Era Variety: 80s legends through current stars with weighted distribution

Multiplayer: Real-time party lobbies for group play

Mobile-First: Optimized for portrait phone screens

Stats & Streaks: Track your performance and climb leaderboards

🛠 Tech Stack
text
Frontend: React + Tailwind CSS
Backend: Supabase (Auth, Database, Realtime)
Deployment: Vercel
🚀 Quick Start
bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/silhouette-hoops-squad.git
cd silhouette-hoops-squad

# Install dependencies
npm install

# Start dev server
npm run dev

# Open http://localhost:3000
